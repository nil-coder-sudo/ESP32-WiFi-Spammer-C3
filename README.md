# ESP32 WiFi Spammer C3
![Build Status](https://github.com/nil-coder-sudo/ESP32-WiFi-Spammer-C3/actions/workflows/build.yml/badge.svg)

A professional WiFi security testing tool for ESP32 and ESP32-C3.

## 🚀 How to use
1. Download the latest firmware from the **Actions** artifacts or **Releases**.
2. Flash the `Full_Flash.bin` to address `0x0`.
3. Enjoy testing!

## 🛠️ Pinout
- **SDA**: GPIO 21 (ESP32) / GPIO 8 (C3)
- **SCL**: GPIO 22 (ESP32) / GPIO 9 (C3)
- **Button**: GPIO 2
